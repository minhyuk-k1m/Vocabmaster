기기 간 데이터 연동 및 백업

실시간 클라우드 동기화 (가입 없음) 별도의 회원가입 없이 6자리 일시적인 동기화 코드를 생성하여 PC와 모바일 간에 로그인 및 데이터를 무설정으로 간편하게 동기화합니다.
파일로 백업 및 복구 (오프라인) 현재 단어장 및 계정 정보를 파일로 다운로드하거나, 이전에 백업한 파일로부터 즉시 복구합니다.

사용 설명서

1 단어장 생성 및 학습

나만의 영단어 목록을 직접 만들고, 카드 뒤집기, 오지선다형 퀴즈, 주관식 테스트를 통해 손쉽게 암기할 수 있습니다.

English Word Card
Challenge
도전, 과제
  • 단어장 생성: 메인 화면에서 [단어장 만들기]를 누르고, 원하는 제목과 영단어/뜻을 한 칸씩 적어 단어장을 등록하세요.
  • 학습하기 (선택형): 뜻을 고르는 오지선다형 퀴즈로, 틀렸을 때는 빨간 X와 함께 오답 피드백이 제공됩니다.
  • 테스트 (주관식): 직접 키보드로 정답을 기입하는 고난도 암기 확인 테스트를 치를 수 있습니다.

2 6자리 코드로 기기 간 연동

어떤 번거로운 가입이나 설정 없이, 6자리 보안 코드를 통해 PC-모바일 기기 간 단어장과 로그인 상태를 똑같이 연동합니다.

PC (내보내기)
────▶
모바일 (가져오기)
  • 내보내기 (기존 기기): 설정 탭에서 [동기화 내보내기 (코드 생성)]를 클릭하면 6자리 코드가 클라우드에 임시 생성됩니다.
  • 가져오기 (새로운 기기): 다른 기기의 설정 탭에서 [동기화 가져오기 (코드 입력)]를 누르고 해당 코드를 입력하면 즉시 로그인이 연동되고 단어장 정보가 온전히 전송됩니다.

3 파일 백업 및 복구 (오프라인)

오프라인 상태이거나 본인의 컴퓨터에 전체 데이터를 영구 백업하고 싶을 때 유용한 JSON 파일 백업 기능입니다.

vocabmaster_backup.json JSON Data File
  • 백업 파일 다운로드: 설정 탭에서 [데이터 백업 파일 저장]을 클릭해 내 컴퓨터에 로컬 백업 파일을 내려받을 수 있습니다.
  • 데이터 복구: 브라우저가 초기화되거나 다른 브라우저에서 [백업 파일 불러오기]를 눌러 다운로드한 백업 파일을 첨부하면 즉시 모든 데이터가 원래대로 복구됩니다.
